# subscene-dl

Downloads subtitles from subscene.com.

## Usage

```bash
subscene-dl [-h] [-t TAGS] language file

Downloads movie subtitles from subscene

positional arguments:
  language              ISO-639-1 (2-letter) or ISO-639-2/B (3-letter)
                        language code
  file                  File name must be in the form 'MOVIE_NAME
                        (RELEASE_DATE)'

optional arguments:
  -h, --help            show this help message and exit
  -t TAGS, --tags TAGS  Filter subtitles by release tags like type and
                        resolution

```
